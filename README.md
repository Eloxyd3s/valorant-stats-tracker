# Valorant Stats Tracker Overlay

A real-time in-game stats overlay for Valorant that displays your performance metrics while playing.

## Features

✨ **Real-time Stats Tracking**
- Kills, Deaths, Assists (K/D/A)
- Round score and economy tracking
- Plant/Defuse stats
- Headshot percentage
- Combat stats

🎮 **Game Integration**
- Auto-detects Valorant window
- Connects to local Valorant API
- Zero-lag overlay rendering
- Auto-start with game launch

⚙️ **Customization**
- Adjustable overlay position and size
- Transparency control
- Color themes
- Font size adjustment
- Show/hide individual stat sections

📊 **Advanced Features**
- Session statistics
- Performance graphs
- Round-by-round breakdowns
- Agent tracking
- Team coordination stats

## Requirements

- Windows 10/11
- Python 3.10+
- Valorant installed
- 100MB disk space

## Installation
### 1. Clone the Repository
```
git clone https://github.com/Eloxyd3s/valorant-stats-tracker.git
cd valorant-stats-tracker
```
### 2. Rune the command in the Repository
```
py -3.11 -m PyInstaller --onefile main.py
```
